# HCLS Artifacts

This repository contains various ADK (Agent Development Kit) agents for Healthcare and Life Sciences.

## Getting Started

This project uses `uv` for dependency management.

### Prerequisites

- [uv](https://github.com/astral-sh/uv) installed.

### Installation

Sync dependencies:

```bash
uv sync
```

## Structure

Each subdirectory represents a distinct agent or component.

- `rxnorm/`: Agent for RxNorm related tasks, configured with Gemini 2.5 Flash.
